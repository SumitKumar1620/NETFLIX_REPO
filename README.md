# Netflix Homepage Clone 🎬

A responsive clone of the Netflix India landing page built using **HTML5** and **CSS3**. This project replicates the modern UI and design of the Netflix homepage, including the hero section, feature sections, FAQ accordion layout, and footer. The main objective of this project was to improve frontend development skills by recreating a real-world website design. The original Netflix homepage features sections such as a hero banner, trending content, feature highlights, FAQs, and responsive layouts across devices.

## ✨ Features

* Responsive design for Desktop, Tablet, and Mobile devices
* Netflix-inspired Hero Section
* Navigation Bar with Sign In button
* Email Subscription Form
* Feature Showcase Sections
* FAQ Accordion Section
* Modern UI with Netflix-inspired styling
* Clean and organized code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries
